# Responsive Personal Portfolio

## Overview

This project was developed as part of my Web Development Internship (Task 2). It is a fully responsive personal portfolio website built using HTML5, CSS3, and JavaScript, focusing on modern layouts, accessibility, and a clean user experience.

## Features

* Responsive design for mobile, tablet, and desktop
* CSS Grid for page layouts
* Flexbox for component alignment
* Light/Dark mode using CSS Variables
* Semantic HTML5 structure
* Accessible navigation and contact form
* Smooth transitions and modern UI
* SEO-friendly structure

## Technologies Used

* HTML5
* CSS3
* JavaScript
* CSS Grid
* Flexbox

## Project Structure

* `index.html` – Home page
* `about.html` – About section
* `projects.html` – Portfolio projects
* `contact.html` – Contact form
* `styles.css` – Styling and responsive layouts
* `theme.js` – Theme toggle functionality

## How to Run

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
